Для запуска проекта следует:
1) Иметь установленные: PHP7.x, MySql-server, MySql-client;
2) Создать локальный сервер (apache + host);
3) Создать базу и в ней выполнить скрипт из файла trees.sql;
4) Сверить свои данные настройки БД с файлом ./config/db.php;
5) Открыть сайт в браузере и проверить работу.
